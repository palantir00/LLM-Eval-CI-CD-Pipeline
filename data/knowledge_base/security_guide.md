# Security Guide

## Unauthorized transactions
If a customer notices an unauthorized transaction on their account, they should block the
card in the app as soon as possible and report the transaction to the bank, for example via
in-app chat or the helpline. This starts the dispute (complaint) procedure.

## Two-factor authentication (2FA)
Two-factor authentication adds a second element of identity confirmation in addition to the
password, for example a one-time code sent by SMS or a confirmation in the app. It makes
unauthorized access much harder even if the password has been stolen.

## Phishing awareness
The bank will never ask the customer for their full password or full card number by email,
SMS, or phone. Customers should treat such requests as phishing attempts and report them.
